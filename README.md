# Operating Systems Lab Examples

## Project Description (English)
This repository contains laboratory examples from the **Operating Systems** course. Each folder represents a separate lab session and includes example programs written in **C** that were implemented during lab hours.

The labs cover fundamental operating system concepts such as process management, threading, synchronization, scheduling, and memory-related topics. These examples were developed for educational purposes to better understand low-level system programming.

---

## Contents
- Each `Lab_X` folder corresponds to a specific lab session
- Each lab folder includes:
  - Example C source files
 
---

## Technologies
- **Language:** C
- **Environment:** Linux / Unix-based systems
- **Tools:** GCC, Terminal

---

## How to Use
- Navigate to any lab folder
- Compile the desired C file using `gcc`
- Run the compiled executable from the terminal

Example:
```bash
gcc example.c -o example
./example

```
